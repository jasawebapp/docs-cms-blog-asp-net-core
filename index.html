Getting Started


Introduction

We would like to thank you for purchasing CMS Blog Asp.net Core! We are very pleased that you have chosen CMS Blog Asp.net Core for your website, you will not be disappointed! Before you get started, please be sure to always search our Documentation. We outline all kinds of good information, and provide you with all the details you need to use CMS Blog Asp.net Core.

Requirements *

  • dotnet-hosting-2.1.6
  • dotnet-hosting-2.2.0
  • dotnet-hosting-2.2.4
  • MSSQL Server / MySql
  • IIS 10


Licenses & Activation

When you purchase a CMS Blog Asp.net Core, you will be granted an inclusive 6 or 12 months - based on your purchase plan option - of hands-on support for that item. When this expires you can choose to extend your item support for a further 6/12 months at a fee. Please continue reading below to learn more about item support, or Click Here To Read Envato’s ‘What Is Item Support?’ Article

Allows you to use license code on one domain only. A domain is considered only the top level domain, you can activate the regular license on multiple subdomains of that domain for example if you activate the code in this domain: www.xyz.com

  • www.xyz.com – Valid
  • subdomain.xyz.com – Valid
  • abc.com – New license is required
  • subdomain.abc.com – New license is required
One license is valid for one domain only. If you want to use the CMS Blog Asp.Net Core on another domain, then you have to purchase another license.

Validate License Code

When your site started, we will apear "Insert License Code" enter your license code inside there like the image below and click install if success you will redirect to installation page, if it is invalid then a "License Code is invalid" popup will appear.



Installation

After licenses code valid you will redirect to installation page like image below



To Install CMS Blog Asp.net Core:

  1. Enter your website name
  2. Enter your Short Name (max length: 12 character)
  3. Select Type Database
  4. Enter your name server database
  5. Enter your Database Name
  6. Enter your Database Username
  7. Enter your Database Password
  8. Enter your email admin
  9. Enter your password admin
after you fill all the inputs click install




When installation success will be show up like below.


Now Login into admin using email and password your input when installation. URL Admin: yourdomain/jw-admin

Settings Configuration


General

This option will help you edit your website name, short name, language site, logo, and set site icon. The Site Icon is used as a browser and pwa icon for your site. Simply go to ‘Settings->General’ and change your website name, short name, logo, and upload your site icon.



Footer

This option will help you edit footer description. Simply go to ‘Settings->Footer’ and change your text, address, phone, email, links.



Custom CSS

Simply go to ‘Settings > Custom CSS’ and write your custom CSS.



Recaptcha

When you create site in recaptcha will generate site key. copy the site key and insert into Site Key Input. Click this link to follow tutorial Create Reacpatcha V2



Web Master

This option will help you edit google verification code, big verification code and google analytics tracking code. Simply go to ‘Settings->Web Master’ and change your google verification code, big verification code and google analytics tracking code.

  • Get your google verification code in Google Search Console
  • Get your bing verification code in Bing webmaster tools
  • Get your analatyics tracking code in Google Analytics


Colors

This option will help you edit colors. Simply go to ‘Settings->Colors’ and change your colors as you want it.



Pages


All Pages

Image below is view table all pages. you can manage it Simply go to ‘Pages->All Pages’.



Add New Page / Post

To add new page Simply go to ‘Pages->Add New’ Then fill in the form and click Save Draft or Publish Button. if you want set page to post just check: "Checkbox Is Post".



Categories

Image below is view table all categories. you can manage it Simply go to ‘Pages->Categories’.



Add New Category

To add new category Simply go to ‘Pages -> Categories And Click Add New Button’ Then fill in the form and click Publish Button.



Tags

Image below is view table all categories. you can manage it Simply go to ‘Pages->Tags.



Add New Tag

To add new tag Simply go to ‘Pages -> Tags And Click Add New Button’ Then fill in the form and click Publish Button.



Home Page

This option will help you to edit home page. Simply go to ‘Pages->Home’ and set title, description, first text, second text, og image, twitter image and text before footer.



Contact Page

This option will help you to edit contact page. Simply go to ‘Pages->Contact’ and set title, description, og image, twitter image and content right side in contact page.
URL Contact Page: yourdomain/contact


Menus

To manage menus Simply go to ‘Menus’


Add By Page

To add menu by page. click pages, check checkbox page you want to add and click add to menu button. don't forget to click save menu button.



Add By Category

To add menu by category. click categories, check checkbox category you want to add and click add to menu button. don't forget to click save menu button.



Add Custom Link

To add custom link into menu. click custom links, fill url and link text and click add to menu button. don't forget to click save menu button.



Add Text with no link

To add text into menu (useful for create dropdown). click Text, fill label text and click add to menu button. don't forget to click save menu button. if you want to create dropdown just drag another menu item below Text Menu Item and check if menu item in view add some margin left that will be generate dropdown.


Roles


All Roles

Click on sidenav Roles & Permissions to look all roles



Add Role

To add some role just fill role name and check the checkbox permission you want to add.


Users


All Users

Click on sidenav 'Users -> All Users' to look all users



Add User

To create new user you just fill username, email, password input and select role user you want to create.


Ads


All Ads

Click on sidenav 'Ads -> All Ads' to look all ads



Add New Ad

To create new ad just fill title input and paste code script adsense into input "Paste AdSense Code To Extract Automatically"



Setting Ad

This option will help you to edit adsense publisher, ad card, block detector. Simply go to ‘Ads -> Setting’ and set your adsense publisher. for ad card if not empty by default ad will be show up between card blog in home, category and tag. to fill add card: add ad first and copy the shortcode in that input. enable or disable ad block detector and change text when user use adblock (if ad block detector enabled).
